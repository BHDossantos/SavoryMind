# FitNow — Product Brief

## Vertical name

**FitNow** — best fit under the larger platform.

Other candidates considered: AvailableNow Fitness, FitSlot, BookFit, TrainNow, WellnessNow, FitOS.

## Core concept

A real-time booking and wellness platform where users can find and book:

- gyms
- fitness classes
- personal trainers
- BJJ / martial arts
- yoga
- pilates
- CrossFit
- massage / recovery
- sauna / cold plunge
- nutrition coaching
- meal prep plans
- workout plans

**Core promise:** Find a workout, trainer, class, or wellness session available now — and book instantly.

## Top 15 problems this solves

1. **People don't know which gyms/classes are available today** — Show real-time availability for classes, trainers, open gym passes, and wellness sessions.
2. **Fitness discovery is fragmented** — Google, Instagram, WhatsApp, gym websites, flyers. One marketplace for local fitness and wellness options.
3. **Tourists and expats need short-term access** — Day passes, weekly passes, drop-in classes, short-term memberships.
4. **Gyms have unused capacity** — Push available slots and last-minute offers.
5. **Trainers struggle to get consistent clients** — Trainer profiles, instant booking, recurring sessions, packages.
6. **Users don't know what workout fits their goal** — Goal-based matching: fat loss, muscle gain, mobility, BJJ conditioning, endurance, recovery.
7. **People lack accountability** — Workout plans, reminders, streaks, trainer check-ins.
8. **Meal prep and fitness are disconnected** — Add meal prep plans tied to goals and workouts.
9. **Users don't know which class is beginner-friendly** — Class difficulty, fitness level, intensity, coach notes.
10. **Booking trial classes is annoying** — Instant trial booking for gyms and studios.
11. **People quit because they don't see progress** — Progress tracking: weight, body fat, measurements, PRs, attendance.
12. **Providers use poor scheduling systems** — Simple provider calendar, capacity management, booking rules.
13. **No-shows hurt gyms and trainers** — Deposits, cancellation policies, reminders.
14. **Users don't know if the vibe fits them** — Tags: hardcore gym, luxury gym, beginner-friendly, women-friendly, combat sports, athletic, social.
15. **Health/wellness services are separate** — Unified platform: train, recover, eat, track.

## Target users

**Customers:** locals, tourists, expats, remote workers, athletes, beginners, busy professionals, people restarting fitness, combat sports athletes, people seeking weight loss/body recomposition.

**Providers:** gyms, personal trainers, BJJ academies, yoga studios, pilates studios, CrossFit boxes, massage therapists, recovery centers, nutritionists, meal prep vendors, sports coaches.

## Core use cases

- Book a workout class today — *"Find me a boxing class tonight near Prati."*
- Book a personal trainer — *"Personal trainer tomorrow morning, fat loss, English-speaking."*
- Buy a day pass — *"Gym day pass near Cipro."*
- Join BJJ/MMA class — *"Beginner BJJ class tonight."*
- Book recovery — *"Sports massage after training."*
- Get a workout plan — *"Create a 4-week fat loss plan."*
- Get a meal prep plan — *"High-protein meal prep under 2,200 calories."*
- Combine services — *"Gym pass + trainer + meal plan."*

## Main app modules

- **Fitness marketplace** — Search and book fitness services.
- **Provider scheduling** — Gyms/trainers manage availability.
- **Class management** — Studios manage classes and capacity.
- **Personal training booking** — Trainers sell sessions and packages.
- **Workout plans** — Users receive structured plans.
- **Meal prep / nutrition plans** — Users receive nutrition support.
- **Progress tracking** — Basic fitness metrics and attendance.
- **Payments** — Deposits, class passes, subscriptions, packages.
- **Reviews** — Ratings for gyms, trainers, classes.
- **Admin panel** — Manage platform operations.

## Customer journey

1. **Open app.** Home asks: *"What do you want to do today?"* Options: gym day pass, fitness class, personal trainer, BJJ / martial arts, yoga / pilates, recovery, workout plan, meal plan.
2. **Select goal.** Lose fat, build muscle, improve cardio, learn martial arts, get stronger, improve mobility, recover, beginner fitness, athletic performance.
3. **Select location and time.** Available now, today, tomorrow, this week, specific date.
4. **App shows options.** e.g. boxing class at 19:00, open gym pass available now, personal trainer at 08:00 tomorrow, sports massage at 21:00, BJJ beginner class at 20:30.
5. **Book.** Service, provider, time, price, cancellation policy, what to bring, difficulty, duration.
6. **Pay.** Full amount, deposit, package credit, or pay at location.
7. **Confirmation.** Booking confirmation, map, instructions, trainer/class details, reminders.
8. **After session.** User rates experience.
9. **Retention.** App suggests: book again, buy package, start workout plan, add meal plan, track progress.

## Provider journey

**Gym/studio setup:** business profile, location, photos, amenities, opening hours, services, class schedule, capacity, trainers, pricing, trial offers, day pass options.

**Trainer setup:** profile, specialties, certifications (optional), languages, availability, session prices, packages, training location, online/in-person options.

**Class setup:** class name, description, coach, date/time, duration, capacity, difficulty, price, equipment needed, cancellation rules.

**Booking management:** accept booking, auto-confirm, cancel, reschedule, mark attended, mark no-show, message customer.

**Provider analytics:** bookings, revenue, attendance, class fill rate, no-show rate, top services, repeat clients, empty slots.

## MVP scope

**Customer MVP:** signup/login, search fitness services, filter by location/time/category, view provider profile, view class/trainer availability, book session/class/day pass, pay optional deposit, view bookings, cancel/reschedule, rate provider.

**Provider MVP:** signup/login, create profile, add services/classes, set availability, manage bookings, mark attendance/no-show, basic dashboard.

**Admin MVP:** approve providers, manage users, manage bookings, manage categories, view platform metrics.

**MVP categories — start with:** gym day pass, personal trainer, BJJ / martial arts, yoga / pilates, boxing / fitness class, massage / recovery.

**Add later:** nutrition coaching, meal prep marketplace, full workout plans, corporate wellness.

## Search and matching logic

**Inputs:** category, goal, location, time, budget, fitness level, distance, language, difficulty, provider rating.

**Matching score:**
- Availability: 30%
- Distance: 20%
- Goal match: 15%
- Fitness level match: 10%
- Rating / reviews: 10%
- Price fit: 10%
- Language match: 5%

**Example.** User wants: *"BJJ beginner class tonight near Prati."* System filters: `category = martial_arts, tag = BJJ, difficulty = beginner, city = Rome, time = tonight, spots_available > 0, distance within radius`. Then ranks by availability, distance, rating, price, language.

## Special fitness features

- **Drop-in pass** — Users buy one-time access.
- **Class capacity** — Show *"12 spots total / 4 spots left."*
- **Waitlist** — If full, join waitlist; on cancellation, notify waitlist.
- **Package credits** — e.g. 5-class pack, 10-session trainer pack.
- **Membership add-on (later)** — Monthly subscriptions for providers.
- **Trainer packages** — e.g. 1 session €60, 5 sessions €275, 10 sessions €500.
- **Goal-based plan recommendation** — After booking: *"Want a 4-week strength plan?"*
- **Meal prep add-on** — After workout plan: *"Add high-protein meal prep."*

## Payment rules

**MVP options:**
- Day pass — pay full amount.
- Class — pay full amount or deposit.
- Personal trainer — deposit or full prepay.
- Massage / recovery — deposit recommended.

**Cancellation policies:**
- Flexible — cancel 4 hours before.
- Moderate — cancel 12 hours before.
- Strict — cancel 24 hours before.

**No-shows.** Provider marks no-show. Rules: deposit lost, user reliability score impacted, repeat no-shows require prepayment.

## Notifications

**Customer:** booking confirmed, class reminder, trainer reminder, class cancelled, spot opened from waitlist, package running low, review request.

**Provider:** new booking, class full, cancellation, waitlist activated, payment received, review received.

## MVP focus recommendation

Don't build everything first. Start with **Rome fitness drop-ins and classes**.

**Best first categories:** BJJ / martial arts, boxing, yoga / pilates, personal training, gym day passes.

**Why:** high repeat, clear schedules, easy to book, strong local communities, tourists/expats want drop-ins, founder personally knows BJJ and gyms.

**Initial MVP supports:** users booking classes / day passes / trainer sessions; providers managing schedules and capacity; payments / deposits (optional); reviews and reminders.

## Launch strategy

Start in **Rome**.

**Target supply:** 20 gyms/studios, 20 personal trainers, 10 BJJ/MMA academies, 10 yoga/pilates studios.

**Initial demand:** expats, tourists, remote workers, BJJ travelers, fitness-focused locals, men/women looking for drop-ins.

**Provider pitch:** *"I can bring you drop-in clients, fill empty class spots, and reduce WhatsApp booking chaos."* — free onboarding, free profile setup, commission only when booking happens, featured placement for early partners.

**Customer pitch:** *"Find and book gyms, classes, trainers, and recovery sessions available today."*

## Monetization

- **Commission per booking:** 10%–20%.
- **Provider subscription:** Basic free, Pro €49/month, Premium €149/month.
- **Featured placement:** €50–€300/month.
- **Packages:** commission on class packs / trainer packs.
- **Meal prep / workout plan upsells:** premium user plans.
- **Corporate wellness (later):** companies pay monthly for employee wellness access.

## Key metrics

**Supply:** active providers, available classes, spots available, provider retention, class fill rate.

**Demand:** searches, bookings, repeat bookings, package purchases, monthly active users.

**Financial:** gross booking value, commission revenue, average order value, provider revenue, refund rate.

**Quality:** ratings, no-show rate, cancellation rate, booking success rate, repeat provider bookings.

## One-sentence dev brief

Build a mobile-first fitness and wellness booking platform where users can discover and instantly book gyms, classes, personal trainers, martial arts, yoga/pilates, and recovery services based on real-time availability, location, goal, fitness level, and price. Providers can create profiles, manage services, schedules, class capacity, bookings, attendance, payments, and analytics. The first version should launch in **Rome** with gyms, trainers, BJJ/MMA academies, yoga/pilates studios, and recovery providers.
